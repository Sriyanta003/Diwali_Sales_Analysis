# Python_Diwali_Sales_Analysis
Python project for beginners- Analyze Diwali sales data to improve customer experience and sales


🪔 Diwali Sales Data Analysis
This project analyzes a Diwali sales dataset using Python to uncover customer purchasing behavior and sales trends. The analysis includes data cleaning, exploratory data analysis (EDA), and visualizations to generate business insights.

📌 Project Overview
The objective of this project is to analyze customer demographics, purchasing patterns, and product performance during Diwali sales to identify key revenue drivers.

📂 Dataset
11,000+ records

Features include: User_ID, Gender, Age Group, Marital Status, State, Occupation, Product Category, Orders, and Amount

🛠 Tools & Technologies

Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

🔎 Key Steps

Data loading and inspection
Handling missing values
Data cleaning and formatting
Exploratory Data Analysis (EDA)
Grouping and aggregations using Pandas
Data visualization for insights

📊 Key Insights

Female customers contributed more to total sales.
Age group 26–35 years generated the highest revenue.
Married customers showed higher purchasing power.
States like Uttar Pradesh, Maharashtra, and Karnataka recorded strong sales.
IT, Healthcare, and Aviation professionals were among the top buyers.
Food, Clothing, and Electronics categories performed well.

📈 Outcome

The project demonstrates practical data analysis skills including data preprocessing, visualization, and business insight generation using Python.
